# FunPay Price Monitor

Автоматизированный инструмент для мониторинга минимальных цен на FunPay с визуализацией данных на рабочем столе через Rainmeter.

## Как это работает
* Скрипт script.pyw работает в фоновом режиме.
* Каждые 5 минут скрипт запрашивает актуальную страницу с сайта FunPay через Playwright.
* Система анализирует список лотов, извлекает минимальную цену и записывает её в файл price.txt.
* Rainmeter считывает файл price.txt и выводит обновляемую информацию прямо на ваш рабочий стол.

## Установка

### 1. Подготовка окружения
Убедитесь, что у вас установлен Python. Установите необходимые библиотеки для работы парсера:
pip install -r requirements.txt
playwright install chromium

### 2. Настройка скрипта
* Скопируйте файл script.pyw в выбранную директорию (например, D:\Funpay).
* Настройте автозапуск через Task Scheduler (Планировщик задач Windows):
    * Action: Start a program
    * Program/script: pythonw
    * Arguments: полный_путь_к_файлу\script.pyw

### 3. Настройка Rainmeter
1. Установите Rainmeter (https://www.rainmeter.net/).
2. Скопируйте папку /skins/FunPayPrice в Documents\Rainmeter\Skins.
3. Обновите Rainmeter и загрузите скин Price.ini.

## Адаптация под другие товары

Чтобы настроить скрипт на мониторинг другого товара, откройте script.pyw и внесите следующие изменения:

### 1. Изменение URL
Найдите строку:
url = "https://funpay.com/chips/209/?server=12605&side=106"
Замените адрес в кавычках на ссылку из адресной строки вашего браузера, ведущую на нужный вам товар.

### 2. Настройка селекторов (фильтрации)
Найдите строку:
items = await page.query_selector_all('a.tc-item[data-server="12605"][data-side="106"]')

* Если товар не имеет привязки к серверу или стороне: удалите часть [data-server="..."][data-side="..."], оставив только 'a.tc-item'.
* Если товар требует другие параметры: откройте страницу нужного товара в браузере, нажмите F12 (инструменты разработчика), найдите нужный лот и посмотрите значения атрибутов data-server и data-side, затем подставьте их в код вместо текущих.

## Поддержка
Если скин на рабочем столе отображает "Loading...", убедитесь, что путь к файлу price.txt, указанный в Rainmeter (файл Price.ini), совпадает с путем, куда скрипт записывает данные.
